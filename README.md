## 🎮 What is castarook?

castarook reimagines chess by adding new game mechanics inspired by role-playing games (RPGs). Instead of only moving chess pieces, you roll dice to unlock special moves or combat options. The game board appears in full 3D, making the chess experience more vivid. It's not just a chess game—it’s a new way to play chess with elements of chance and skill.

### Key Features

- 3D chess board with clear, low-poly models  
- Dice-based combat affecting piece moves  
- RPG-style progression between games  
- Procedural generation of some game elements  
- Smooth controls built with React and Three.js  
- Runs on Windows using WebGL for graphics

## 🎯 How to Play castarook

castarook keeps the basic rules of chess but adds dice rolls to change how pieces move or fight. Here is how to start playing:

### Basic Controls

- Use your mouse to select a piece on the 3D board.  
- Click again to move the piece to a valid square.  
- Dice appear when a combat or special move is possible. Roll dice by clicking on them.  
- Watch the results to see how combat or moves change.

### Game Flow

- Players take turns moving pieces like normal chess.  
- Some moves unlock combat phases or special abilities based on dice rolls.  
- The board and pieces are rendered in 3D, so use your mouse or scroll wheel to look around.  
- Win by checkmating the opponent’s king or through special RPG conditions.
