# Project Overview

This project is about a chess game with RPG elements.
The game will have a regular chess stuff.

But every piece will have unique RPG D&D attributes.
For example a PAWN will have experience that will be used with dice.

Every attack/taking piece will be by both attacker and defender rolling dice.
Attacker piece will have dice + number of kills of the attacker
Defender piece will have dice + number of defended count

Like regular chess make every pieces possible moves and possible takable pieces.

it will be multiplayer and also against ai.

Use react + webgl + three.js.
